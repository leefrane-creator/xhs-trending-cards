# XiaoHongShu Copywriting Guide for Trending Cards

## Title Formula (High CTR)

Pattern: `[Emotion Word] + [Number] + [Topic] + [Result/Hook]`

Examples:
- "炸了！本周5个AI项目暴涨100万Star，开发者都在用"
- "GitHub疯了！这5个开源项目直接封神，第3个我每天用"
- "别再问我用什么AI了！这5个开源神器免费又好用"
- "2026最值得关注的5个AI项目，错过真的会后悔"
- "程序员必备！这5个GitHub项目让效率翻10倍"

Key principles:
- Lead with emotion (炸了/疯了一定要/别再问)
- Include specific numbers (5个/100万Star/Top1)
- Create FOMO or curiosity gap
- Keep under 20 characters for mobile display

## Body Structure

```
[Hook - 2 sentences setting context]
[Item 1 paragraph + link]
[Item 2 paragraph + link]
[Item 3 paragraph + link]
[Item 4 paragraph + link]
[Item 5 paragraph + link]
[Summary/Closing - 1-2 sentences with call to action]
```

Per-item format:
> **#排名 项目名** — 一句话定位
> 2-3句介绍核心价值+亮点。为什么值得关注？解决了什么问题？
> 🔗 github.com/org/repo

## Tag Strategy

Format: 20 tags, mix of broad + niche:

**Broad reach (8-10)**:
#GitHub #开源 #AI #程序员 #开发者 #工具推荐 #效率 #技术分享

**Niche targeted (6-8)**:
#AI助手 #大模型 #Agent #LLM #本地部署 #自动化

**Trending/hype (4-6)**:
#2026趋势 #科技前沿 #开源神器 #程序员必备 #AI工具

## Publishing Tips

| Factor | Recommendation |
|--------|---------------|
| Best time | Tue-Thu 7:00-9:00 PM or 12:00-1:00 PM |
| Image order | Cover first, then detail cards in rank order |
| Image count | 6-7 images (cover + details) performs best |
| Caption length | 300-500 characters optimal |
| CTA at end | "收藏不迷路" / "关注看更多" |
| First comment | Pin a resource summary or GitHub link list |

## Tone Guidelines

- Conversational but authoritative ("我用了一个月" > "据悉")
- Use emojis sparingly (1 per item max)
- Avoid marketing-speak ("颠覆性""革命性" → say具体好处)
- Include personal experience when possible ("亲测好用")
